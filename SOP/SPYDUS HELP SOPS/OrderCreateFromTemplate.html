    Create a new order from a template
    In This Topic

    Rather than filling in all the details manually every time you create an order, you can set up the details in an order template and then create an order based on this template.

    In Acquisitions

    To create a new order from a template in Acquisitions:

    1. From the Module Menu select Acquisitions.
    2. Select New and then Order from Template, or press the F7 function key.
    3. This lists the templates. Do one of the following.
      • Click to select the template you want, and then click OK.
      • Click Search Template, find the template you want and then click Select Record.
    4. Fill in as much detail as required.
      • Bibliographic details
      • Order details
      • Continuous order details
      • Funding and allocation
      • Claims and reminders
      • Reservations
      • Requestor
    5. Click Save & Close to go back to the Order List.

    You can click a column heading to sort the list of templates in ascending or descending order.

    You can click  to filter the list to show only the templates with a particular ID, type, name, status or vendor.

    By default we only show the active order templates, i.e. only those with active funds, but you can click Include inactive to show all templates if you wish.

    You can click Refresh Templates to update the order templates. You might want to do this if the funds on the templates have been changed (for example, through a Fund rollover) on a different computer.

    Troubleshooting

    • Vendor
      • If the Vendor in a MARC file is different from the Vendor specified in the template, the user will be prompted to confirm the Vendor
    • Allocation
      • For each title in a MARC file, if there is a usable 960 tag for Allocation, then that tag will be user to create the Order (LI record) Allocation, and the Allocation details from the template will not be used
      • For each title in a MARC file, if there is no usable 960 tag, then the Allocation details from the template will be used
      • If no suitable Allocation details are found in either the MARC 960 tag or the template, no allocation will be created for the Order (LI)
    • Currency
      • Individual Order (LI) Currency is determined in the following order of precedence: 960 tag > Template > Vendor default > system default
      • Currency for the Purchase Order (PO) is determined in the following order of precedence: Template > Vendor default > system default
    • Other details
      • Other information that is included in the template (and not in the 960 tag) will be applied to the Order (LI) records as with manual Order creation (e.g. Service charges, Notes, etc)

    In Enquiry

    To create a new order from a template in Enquiry:

    1. From the Module Menu select Enquiry, Acquisitions and then Order Template.
    2. Search and find the template you want. At the Full Display, click Create Order.

     

    1. Fill in as much detail as required.
    2. Click Save & Close to go back to the Order List.
