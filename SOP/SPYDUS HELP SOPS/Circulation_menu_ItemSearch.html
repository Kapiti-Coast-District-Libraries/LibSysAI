    Item search
    In This Topic

    Use the Item Search option to search for items that meet certain search criteria. For example, you might choose to search for all items onloan in a particular collection, or all items with a first available date of “thismonth”.

    • First available date. The date when the item was first available for loan. This is the date when the item was first returned. For example, you receive the item, accession it, print the spine label, and then you return it at the Front Desk to remove the final copy tracking status.
    • Last stock rotate date. The date when the item was last moved from one stock rotation location to another.
    • Due for binding. Use this to find items that are due for binding. These usually relate to Serials issues and have to be set up in Item Maintenance.
    • Digital item. Use this to find items that are in digital format.
    • Changing item statuses. You can set or remove an item status through Staff Enquiry. Go to Circulation > Item Search and search for the item you want. Then click Change Item Status. Select the status you want to set or remove, and the click Set Status or Remove Status.

    You can only set or remove the “simpler” item statuses through the Web, not the more “involved” statuses such as Reported Lost, Lost or Damaged. Those can only be set or removed through Circulation. 

    Change item status option is not displayed when a user belongs to a user group that is branch restricted and the home location of the item does not match their user account location. If the user account has no location then the option will be displayed.

    Call number

    Use this option to search the number or part of the number in a Call number or Class number

    Available for allocation

    You can use the Available for Allocation option to search for items that are available for satisfying reservations. For example, you might choose to report on the items that are available for allocation at your branch, sort the results by call number, print the Instant Report and then use this to find the items on the shelf.

    • Typically you’d set the Location to be the current location.
    • By default, only the collections that allow reservations are selected in the Collection box, i.e. only collections with Allow reservations set to Yes. You can use SHIFT+click or CTRL+click to select several locations. Use CTRL+click to deselect a selection.
    • By default, the item statuses that prevent reservations are selected in the Excluding status box, i.e. item statuses with Allow reservations set to No and/or Block allocation set to Yes.
    By default, the reservations are sorted by collection and then call number.

    Available for profile allocation

    Use this option to search for items that are available for satisfying reservations placed through profile matching.

    • Typically you’d set the Location to be the current location.
    • By default, only the collections that allow reservations are selected in the Collection box, i.e. only collections with Allow reservations set to Yes. You can use SHIFT+click or CTRL+click to select several locations. Use CTRL+click to deselect a selection.
    • By default, the item statuses that prevent reservations are selected in the Excluding status box, i.e. item statuses with Allow reservations set to No and/or Block allocation set to Yes.
    By default, the reservations are sorted by collection and then call number.

    Shelf check assigned

    Use this option to search for items that are shelf check assigned.

    Deleted item search

    For statistical and data integrity reasons once an item has been received it cannot be removed from the database. So when we “delete” items we really “deactivate” them, i.e. the items are retained in the database but removed from the active view.

    Deactivated items can still be viewed in the Web display via navigation links, for example from a borrower’s previous loan record, and via Deleted Item Search.

    Item not loaned search

    Use this option to create an instant report to show the items that have not been borrowed within a specified period.

    You must indicate the date or date ranges in the Exclude Issue Date field.

    You can use the format “>01/01/2016” or “1/01/2015 – 31/12/2015” or any of the date search options such as “lastyear” or “thismonth”.

    It is also possible to narrow by location, collection, status or download provider (for resources loaded using the load controls set up for eResource interfaces).

    Once you have a list of records, it might be useful to refine the search to items first available after a certain date, to eliminate items on order. So a suggested search would be Exclude Issue date: >1 may 2012 and Location: Abbey Library. This will find any items at Abbey branch not borrowed since 1 May 2012. Then refine your search by adding First Available date: <1 may 2012. This will narrow your search to only include items received before 1 May 2012 and not borrowed since.
