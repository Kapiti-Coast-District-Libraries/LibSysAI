    Spydus OPAC features in 2025R1
    In This Topic

    Significant changes to OPAC SavedList functionality

    A number of changes have been made to improve SavedList functionality for borrowers at the OPAC.

    'SavedList' term change and configurability

    Following consultation with libraries, the default name for SavedLists at the OPAC is now 'My Picks', but is also configurable. This is an option under the SavedLists heading in the Menu options section of the My account tab of the OPAC HTML Configurator.  

    If Enabled, the plain-text Label may be configured as desired (e.g. My Shelf, My Lists etc), as well as selecting an Icon from the available FontAwesome pack, and the tooltip displayed when mousing over the menu option.

    The Label will displayed in the dashboard menu, the borrower dropdown menu, and will be appended to the the 'View...' and 'Manage..' buttons.

    Gallery display view for borrower SavedLists at OPAC

    Clicking the 'My Picks' link in the borrower menu will display borrower SavedLists in gallery display containers, with the option to manage & edit the lists.

    images/2025R1_OPAC_SAVEDLIST_GALLERIES_thumb.png

    Click image to enlarge

    Clicking the Manage My Picks button gives borrowers options to:

    • create a New SavedList
    • click the ellipsis at the end of the row to Delete a list or Edit details of the list (the Title and optional Description)
    • click View Picks to return to gallery page

     

    The SavedList section on the My account tab of the OPAC HTML Configurator allows staff to configure:

    • how many SavedLists per page will be displayed (page navigation will be displayed if the borrower has more lists)
    • the Container size determines the size of the SavedList galleries
      • currently this only controls the size of cover images returned by an enrichment service
    • the Max records to display in the gallery (the number of records that may be scrolled through in the display) - clicking Show All will display all records in the list regardless of this parameter

    Add ability for borrowers to move or copy records between SavedLists

    When viewing the records of an individual SavedList, a new option has been added to the Options menu to Move/Copy selections or Move/copy all.

    Click the desired option to choose to the list that the records will be copied/moved to, and set the 'Remove records from the original...' checkbox as required.

    Checking this box will move the records (removing them from the original list), while leaving it unchecked will add the records to the new selected list while also leaving them in the original list.

    Consolidate display of SavedLists in borrower dropdown menu

    To ensure usability in the case of borrowers who have a lot of SavedLists, the SavedLists heading in OPAC borrower dropdown menu now displays:

    • the currently active SavedList ('Tolkien' in the below screenshot)
    • the Change active option, hovering over this menu option will pop out a sub-menu of the borrower's other SavedLists allowing the setting of another list to active
    • the View all link to the gallery display page 

    Set the default format when downloading or emailing SavedLists

    It is now possible for staff to set the default Format for SavedLists when downloaded or emailed from the OPAC. Available Format options are:

    • Citation
    • Full
    • RIS Reference Format

    Borrowers may still choose their preferred format from the dropdown list.

    The Default format to export SavedList may be set in the SavedList section on the General tab of the OPAC HTML Configurator.

     Set maximum records Emailing/Downloading a SavedList

    It is now possible for staff to set the maximum number of records that may be exported in a SavedList. This parameter can be used to ensure that long-running processes to generate large lists do not impact system performance.

    Navigate to Maintenance > General > Staff General Parameters > SavedList to set the Maximum number of records exported in OPAC.

    Allow scheduling of OPAC announcements

    The Announcements section of the OPAC HTML Configurator now supports the scheduling of announcements. Announcements now optionally support adding a Start date and Expiry date.

    Announcements with a Start date will display from midnight at the beginning of the day (i.e. 00:00:00).

    Announcements with an Expiry date will display until midnight at the end of the day (i.e. 23:59:59).

    Implement app-based multi-factor authentication for borrowers 

    Spydus now supports the use of authenticator apps for OPAC multi-factor authentication (e.g. Microsoft or Google Authenticator, Authy).

    If a borrower enables (or has already enabled) 2-factor authentication, the option to Manage authenticator app will be available from the borrower menu in the top-right when logged in at the OPAC.

    Selecting this option and clicking Enable will generate a QR code & Secret Key, either of which may be used to configure Code generation in their choise authenticator app.

    images/2025R1_OPAC_MFA_ENABLE_QR_thumb.png

    Click image to enlarge

    After enabling multi-factor authentication and logging out, on next login the borrower will be prompted for any authentication methods that are supported both by the library and the borrower's registration details. For example, if:

    • if the library supports all MFA methods (email, SMS & app)
    • borrower has enabled 2FA at OPAC
    • borrower has not configured an authenticator app
    • borrower has their email address in registration details
    • borrower does not have their mobile phone number in registration details

    ..then the borrower will be prompted to provide an authentication code delivered by email. If multiple authentication methods are supported the library & borrower, the borrower may select which method they would prefer to use.

    images/2025R1_OPAC_MFA_SELECT_METHOD_thumb.png

    Click image to enlarge

    See the Spydus Maintenance features in 2025R1 page for more information on configuring this functionality.

    If a borrower has configured app-based authentication and needs to clear & reset the device being used (e.g. lost or stolen phone), staff may use the Clear Auth App button in Circulation borrower registration to allow the borrower to access their account online.

    Add Local Notes to OPAC bibliographic record display

    A number of libraries had previously had commissioning performed to display information stored in MARC Local Note fields at the OPAC, and this commissioning was been lost in the upgrade to Spydus 11. 

    The Display Parameters for View code OPAC > ID code BIB > Type code FULL_FULL now include 'Local Notes from tag...' for tags 590-598.

    To display the content of these tags at the OPAC, set the Enabled property to Yes, and apply an appropriate Caption for the field content.

    Add Work record format icon

    Previously, no record format icon was displayed on Work records. The FontAwesome icon selected for the Work format in the Record format icon section of the OPAC HTML Configurator will be displayed in the top left of the cover image (highlighted below).

    Restrict borrower account renewal at OPAC based on borrower category

    A new parameter has been added to Borrower Categories Maintenance to allow or disallow borrowers from renewing their account at the OPAC (previously the feature was either enabled or disabled for all borrowers).

    A configurable message may be displayed at the OPAC Personal Details page if a borrower is within the Registration expiry cutoff period (or registration has already expired) and renewal is not allowed.

    See the Spydus Maintenance features in 2025R1 page for more information on configuring this functionality.

    Add thumbnail icons for 'New Titles' and 'Previously borrowed' materials in OPAC

    Library staff can configure the OPAC to include icons on the cover image of 'New titles' and 'Previously borrowed' materials (current functionality only displays configured text on the title if enabled).

    See the Spydus Maintenance features in 2025R1 page for more information on configuring this functionality.

    Display available languages in OPAC footer

    Library staff can now enable functionality for OPAC visitors to switch between any installed language packs with a pop-up menu in the OPAC footer.

    See the Spydus Maintenance features in 2025R1 page for more information on configuring this functionality.
