    Spydus General features in 2023R3
    In This Topic

    Allow record selection over multiple pages in Enquiry & OPAC

    Spydus now maintains the selection of records across multiple pages of search results in both Enquiry and at the OPAC. A numerical badge with a tick will indicate how many records (across all pages) are selected.

    Mousing over the badge will turn the tick to a cross, allowing the user to deselect all currently selected records.

    Ensure title bar consistency

    The tab header in web browsers should now display consistently, either;

    • ModuleName - Spydus, or
    • SubmoduleName - ModuleName - Spydus

    e.g. 'Home - Spydus', 'Returns - Circulation - Spydus'

    This feature requires server version 10.7.1 or higher.  

    Add File Manager functionality to URL link fields

    This feature requires license version 11.4 or later.  

    Staff will now be able to link images from 'File Manager'. An additional 'Select' button is displayed adjacent to URL Link fields and within all rich text controls when the 'Insert image' button is selected.

     This can be found in:

    • Container Parameters > Container > Image
    • OPAC HTML Configurator > General > Logo>Navigation bar logo URL
    • OPAC HTML Configurator > General > Sharing> Image URL for Facebook shares
    • OPAC HTML Configurator > CSS Styles > Header background image
    • Register Online Parameters > Logo URL

     Clicking on this 'Select' or 'Insert image' button opens the File Manager popup, allowing users to select or upload a file.

    When inserting images, staff can select the images uploaded to File Manager. Clicking 'Insert' will add the image into the editor.

    Using the 'Select' button, allows staff to choose a file, click 'Select', and it will paste the URL to the file into the textbox.

    Add 'from' email address option to eVendor harvester command lines

    This feature is available to all Spydus versions with the minimum release:

    • W10.9G for Spydus 10 libraries
    • 2023R3 for Spydus license or subscription libraries

    It is now possible for eVendor harvester notification emails to be sent from a custom address. Setting this will override the default 'harvester_donotreply@spydus.com' address.

    Please contact Civica Support for assistance as commissioning is required.
