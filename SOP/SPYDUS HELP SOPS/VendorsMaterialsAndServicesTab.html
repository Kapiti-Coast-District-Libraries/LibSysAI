    Materials and service charges for a vendor
    In This Topic

    On the Materials and Services tab you can specify the discount for different material types, and the charge for different service types. These are then applied to orders for the selected vendor.

    Materials

    To add, edit or delete the discount rate for a material:

    1. From the Module Menu select Maintenance, Acquisitions and then Vendors.
    2. Do one of the following.
      • Type the Code for the vendor you want, and press Tab.
      • Click Search, find the vendor you want, and click Select Record.
    3. Click the Materials and Services tab.
    4. Under Materials, do one of the following.
      • Click Add to add a new material, and set the parameters you want (see below for details).
      • Click Edit to make changes to the parameters for an existing material.
      • Click Delete to delete a material. 
    5. Click Save or Save & Close.

    Material

    When you select the material type in an order, Spydus will check if that material type has been set up for the vendor. If it has then the discount rate, tax type and tax rate for that material will be automatically applied.

    The valid material codes and descriptions are set up in the VMT lookup table, and can be maintained using the Table Manager.

    Discount

    The discount rate for the selected material type.

    Tax type and tax rate

    By default the Tax type and Tax rate for each material will be the same as those specified on the General tab, but you can change this is you wish.

    Service charges

    To add, edit or delete the discount rate for a material:

    1. From the Module Menu select Maintenance, Acquisitions and then Vendors.
    2. Do one of the following.
      • Type the Code for the vendor you want, and press Tab.
      • Click Search, find the vendor you want, and click Select Record.
    3. Click the Materials and Services tab.
    4. Under Service Charges, do one of the following.
      • Click Add to add a new service charge for a material, and set the parameters you want (see below for details).
      • Click Edit to make changes to the parameters for an existing service charge.
      • Click Delete to delete a service charge. 
    5. Click Save or Save & Close.

    Material

    The valid material codes and descriptions are set up in the VMT lookup table, and can be maintained using the Table Manager.

    Service

    Once you've selected a material, you can specify a service type. You can add many different service types for each material.

    When you select the service type in an order, Spydus will check if that service type has been set up for the selected material type for the vendor. If it has then the service charge will be automatically applied.

    The valid service codes and descriptions are set up in the VSC lookup table, and can be maintained using the Table Manager.

    Charge

    The charge for the service type.

    Tax type

    Specifies whether to apply tax to the service charge.
