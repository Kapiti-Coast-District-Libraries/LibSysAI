    Authority fields and bibliographic fields
    In This Topic

    MARC authority fields are different from those used in bibliographic records.

    An authority record in Spydus will contain some of the following fields.

    • IRN — Internal record number.
    • Leader.
    • 001 — Unique identifier. This is system-generated.
    • 003 — Owner of the record. This is system-generated.
    • 005 — Date and time of last transaction. This is system-generated.
    • 008 — Fixed length data elements including record type (for example, whether it can be used as a subject, name etc.), thesaurus and type of series. This is system-generated.
    • 1XX — Heading. You’ll find more information about headings in the table on page 6 below.
    • 360 — Complex reference – subject.
    • 4XX — See from reference.
    • 5XX — See also from reference.
    • 6XX — Notes.
