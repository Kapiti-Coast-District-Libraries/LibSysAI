    Workflow
    In This Topic

    As of Spydus server version 10.2, Workflows have been deprecated and replace with the Home module and widgets.

    You can use the Workflow Manager to notify staff of tasks to be performed, for example comments to be reviewed or shelf check assigned items to be allocated. And via the Workflow Manager you can complete those tasks, for example, approve or decline comments, or complete shelf check assignments.

    In the user account you specify the group you belong to, and in the group you specify the workflow roles (i.e. the tasks to be performed).

    Roles for reviewing comments and reviewing shelf check assignments are supplied as standard, but you can create your own roles in the workflow parameters.

    When you select Workflow from the Module Menu, you'll see the roles specified for your user group.
