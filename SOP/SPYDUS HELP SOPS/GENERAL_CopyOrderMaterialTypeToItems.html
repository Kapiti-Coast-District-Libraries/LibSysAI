    Copy order Material type to items
    In This Topic

    If set to Yes, the Material type specified on the order will be inherited to items that are created when the order is Confirmed.

    The Material type drop-down menu is populated by the unlocked Material Code (VMT) table in Table Manager.

    This feature requires server version 10.5.1 or higher.
