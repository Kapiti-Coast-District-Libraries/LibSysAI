    COVID-19 Quarantine Factsheet
    In This Topic

    As libraries re-open following restrictions related to coronavirus, naturally there are concerns around the potential exposure and transmission by contact of library materials. A number of new parameters have been introduced to Spydus, available as of the W10.5C WBA update, to assist with the quarantining of materials following return from loan.

    Ensure that any quarantine processes are fully tested in your library's TEST system before committing those changes to the LIVE system.

    Circulation General Parameters

    Two new parameters have been added to the Circulation General Parameters to facilitate the automatic application of a selected status.

    Status to apply at return

    Selecting an Item status from the dropdown menu for the Status to apply at return parameter will automatically apply the selected status when an item is returned from loan. This should be set to affect items coming in from outside the library.

    Status to apply at inhouse return

    Selecting an Item status from the dropdown menu for the Status to apply at inhouse return parameter will automatically apply the selected status when an item is returned via the Circulation returns interface, but is not on loan. This parameter should be set to apply the selected status for items that may have been returned from loan before these parameters were set, but still require quarantine.

    A different Item status may be used for each parameter, and each status may be configured differently.

    e.g. inhouse returns status may be automatically removed after 48 hours, but the status for items being returned from loan remains for 72 hours.

    Item Status Parameters

    A new parameter has been added to the General section of Item Status parameters to assist with the automatic removal of a status.

    Period when status automatically removed

    If defined, this parameter will be applied with the execution of a scheduled task on the Spydus server. Specify an integer - in hours - for the desired quarantine period (e.g. 48, 72, 120, etc).

    This parameter does not support decimal values, only whole integers.

    If this value is not defined, a status will not be targeted for removal by the Spydus scheduled task. Conversely, if a status is not being removed as expected, contact the Civica Service Desk to confirm that the scheduled task is configured and running.

    Recommended configuration of Item Status parameters

    These parameter recommendations are based on the idea that we want to eliminate circulation of items while quarantined, but allow future access to them. Your library may operate under different conditions and prefer a more strict, or more flexible, arrangement.
