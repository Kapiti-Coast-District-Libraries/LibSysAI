    Global reservations
    In This Topic

    Global reservations active

    To specify whether items from any institution can be selected to satisfy a reservation:

    1. From the Module Menu select Maintenance, Circulation and then General Parameters.
    2. Set Global reservations active to Yes or No.
    3. Click Save or Save & Close.

    Global reservations - allow other institutions to edit

    To specify whether other institutions can edit or cancel reservations for pickup at this institution:

    1. From the Module Menu select Maintenance, Circulation and then General Parameters.
    2. Set Global reservations - allow other institutions to edit to Yes or No.
    3. Click Save or Save & Close.

    Global reservations - select institution first

    To specify whether you're prompted to select the institution in an OPAC search across the network:

    1. From the Module Menu select Maintenance, Circulation and then General Parameters.
    2. Set Global reservations - select institution first to 0, 1 or 2.
      0 No. You won't be prompted to select the institution.
      1

      Yes. Before the Place Reservation page is displayed, you'll be prompted to select one of the institutions that has reservable holdings. On the Place Reservation page you'll only be able to select a pickup location or item from the selected institution.

      2

      Yes. Before the Place Reservation page is displayed, you'll be prompted to select one of the institutions that has reservable holdings or any item held in the network. On the Place Reservation page you'll be able to select a pickup location or item from the selected institution (if you selected an institution) or any pickup location or item (if you selected any item held in the network).

    3. Click Save or Save & Close.
    Null means the same as No.
