    Enquiry HTML Configurator - Global/Consortium
    In This Topic

    The Global/Consortium parameters control the default behaviour of global searches in the Enquiry module.

    Global/Consortium parameters apply only if a library is operating as part of a consortium.

    To configure these parameters:

    1. Navigate to Maintenance > OPAC & Enquiry > Enquiry HTML Configurator.
    2. Configure parameters as required.
    3. Click Save or Save & Close.

    Network description

    The Network description is a text label for the name of the consortium, that will be appended to to global expansion option at the Brief Display of search results.

    Enable global expansion

    If selected, staff users will be able to expand their search from covering a single institution to covering all institutions within the consortium. The option to expand the search is available from the Brief Display of a set of search results. 

    Set global search as default

    If selected, when a staff user performs a search, the default results will include all institutions within the consortium. The option will be provided to then narrow the search to results from only the local institution.

    Use local table lookups on global search pages

    If selected, local table lookups will be used on search/refine pages when consortium search is active.
