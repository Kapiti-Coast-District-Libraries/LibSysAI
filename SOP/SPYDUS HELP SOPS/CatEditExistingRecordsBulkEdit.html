    Bulk edit existing bibliographic records
    In This Topic

    The Cataloguing Bulk Edit feature allows users to modify sets of Spydus bibliographic records from within the web based app.

    Previously, to bulk edit Spydus bib records, a user would need to;

    • search Spydus for the desired record set,
    • export the bib records to a MARC file,
    • open the MARC file in the Cataloguing module,
    • use the Select/Change feature to update the records,
    • re-load the records into Spydus

    Bulk Edit has streamlined that process.

    1. Navigate to Cataloguing > MARC > Bulk Edit.
    2. Click the Add to List button in the action bar.
    3. Perform a Bibliographic Search to return the record set.
      • Click Select Set at the Brief Display to add the records to the Bulk Edit list.
    4. From the Edit menu, click the Select/Change option.
      images/CATALOGUING_MARC_BULK_EDIT_LIST_thumb.png

      Click image to enlarge

    5. Use the Select/Change function to select which fields/subfields/indicators etc to modify, and how they are to be changed.
    6. Validate the changed records to ensure that they remain consistent with MARC standards (Tools menu > Validate Record(s) or Validate All Records).
    7. Click Save All to commit the changes to Spydus.
    For more detailed instructions on using the Select/Change feature, please review this article.
