    Movement for Archives
    In This Topic

    You can track the movement of accession records, physical items and containers (storage locations). The history of movements is kept on each record.

    It is not possible to move DIGITAL items, only PHYSICAL items. If a physical item group is moved, then any child records are automatically moved and a movement record is created for each child record.

    First we'll move the item and then receive it at the new storage location.

    Moving an item

    To move an item:

    1. Search for the accession record, physical item or container you want to move.
    2. Click Move at the Full Display to open the Movement Editor.
    3. Click Search to find the storage location you want, and click Select Record at the Full Display.
    4. You can select Permanent move if you wish.
    5. Set the status — "Pending" or "Intransit".
    6. Specify a reason if you wish.
    7. Click Save & Close. This creates a movement record.

    Receiving an item

    Now to receive the item at the storage location.

    1. Do one of the following.
      • Search for the accession record, physical item or container you moved, and click Edit Movement at the Full Display.
      • Search for the movement, and click Edit Record at the Full Display.
    2. Set the status to "Received".
    3. Click Save & Close.

    If the item was already linked to an original storage location when it moved you'll now see both a Storage location (the original location) as well as a Current location on the Full Display. If the item was not already linked to an original storage location you'll only see a Current location on the Full Display.
