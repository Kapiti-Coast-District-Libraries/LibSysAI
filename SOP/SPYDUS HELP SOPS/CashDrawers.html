    Cash drawers
    In This Topic

    Specify the cash drawer device and model

    To specify the device and model, click the User Menu at the top right of the screen and then click Preferences.

    Set up the drawers

    To set up the cash drawers:

    1. From the Module Menu select Maintenance, General and then Table Manager.
    2. Select the DRW lookup table and add, edit or delete the drawers you want.
    3. Click Save or Save & Close.

    Set up the payment modes

    To set up the payment modes:

    1. From the Module Menu select Maintenance, General and then Table Manager.
    2. Select the ZCJ lookup table and add, edit or delete the modes you want, e.g. cash, cheque and credit card.
    3. Click Save or Save & Close.

    The modes that kick open the drawer and the modes that have rounding applied are specified in the payment mode parameters.

    Set up the denomination types

    To set up the denomination types:

    1. From the Module Menu select Maintenance, General and then Table Manager.
    2. Select the DNM lookup table and add, edit or delete the types you want, e.g. note and coin.
    3. Click Save or Save & Close.

    Set up the denominations

    When you close or edit a cash drawer you can choose the option to Use denominations, i.e. rather than simply specify the total amount in the drawer, you can breakdown the amount by coins and notes.

    To specify the denominations you can use:

    1. From the Module Menu select Maintenance, Circulation and then Cash Drawer Denominations.
    2. Click Add to add a new denomination. Click Edit or Delete to edit or delete an existing one. The denominations will be displayed in the order they're listed here. You can use Move Up or Move Down to change the order.
    3. Click Save or Save & Close.

    Set up the general cash parameters

    To set up the general parameters for a cash drawer:

    1. From the Module Menu select Maintenance, Circulation and then General Parameters.
    2. Set up the cash drawer parameters you want.

    Use cash drawer

    This specifies whether a cash drawer is in use and if it is mandatory.

    Enable cash drawer options for generic staff

    This specifies whether the cash drawer menu options are enabled when you're logged in as a generic user. If this is set to No then normal staff privilege validation is applied. If this is set to Yes then all options will be enabled but generic staff will be prompted for additional staff credentials when they select a cash drawer option. Note the additional staff prompt will not accept generic staff credentials.

    Prompt whether to close cash drawer on logout

    This specifies whether you're prompted to close a cash drawer when you logout.

    1. Click Save or Save & Close.
    Null means the same as No.  

    Connect to a drawer

    To open a cash drawer:

    1. From the Module Menu select Circulation, Cash Drawer and then Connect to.
    2. Do one of the following.
    • Select Select currently open cash drawer, select the one you want from the list and click OK.
    • Select Open a new cash drawer, select the one you want from the list, enter an opening balance and click OK. Enter your ID and password and click OK.

    Close a drawer

    To close a drawer that's currently open:

    1. From the Module Menu select Circulation, Cash Drawer and then Close.
    2. Enter your ID and password and click OK.
    3. This lists all the drawers that are currently open. Select the one you wish to close.
    4. Do one of the following.
    • Tally the amount in the drawer and enter this in the Actual box.
    • Select Use denominations and then enter the quantity of each denomination.
    • Select Use separate totals if you want to enter separate totals for cash, cheque and cards.
    1. Click OK. If the Actual amount is different from the Expected amount then you’ll need to enter a Closing note before you can click OK.

    Edit a closed drawer

    To change the closing balance for a drawer that's been closed:

    1. From the Module Menu select Circulation, Cash Drawer and then Edit.
    2. Enter your ID and password and click OK.
    3. Search and find the drawer you want, and click Select Record at the Full Display.
    4. Make your changes and click OK.

    "No sale" option

    You can select Circulation, Cash Drawer and then Kick Drawer to create a "no sale" receipt. You can search for these receipts using No sale option on the Receipts search page. On the Full Display for the receipt you'll see the User who selected the Kick Drawer option. If a cash drawer is connected you'll see a link to the drawer session too.
