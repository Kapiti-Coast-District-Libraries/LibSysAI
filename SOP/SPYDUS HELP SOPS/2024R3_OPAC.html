    Predefined values in Borrower Join Online forms

    When selecting a form type in Maintenance > OPAC & Enquiry > Borrower Join Online, two new options are available in the Field properties section:

    • Preset value field (or Checked by default for checkbox fields)
    • Hidden checkbox
    images/2024R3_OPAC_BRWREG_PRESET_thumb.png

    Click image to enlarge

    It is now possible to set pre-defined values in the fields of borrower registration forms at the OPAC, and to have fields with pre-defined values hidden from the registration forms. This feature accomodates changes previously made to XSL files by library staff to allow hidden form values to be set.

    Some examples of use cases might be:

    • Enabling the loan history check flag in a registration form by default and displaying the field to allow borrowers the opportunity to disable it
    • In school libraries, set the year level, home group etc for a given registration form and hide the field so that borrowers cannot change it

    Preset values and hidden flag are not supported for the following fields:  

    • First names (NAM_FNAMES)
    • Surname (NAM_SURNAME)
    • Borrower ID (BDN_ID)
    • Username (UNM_ID)
    • Secondary Id (SBI_ID)
    • Email address (BDE_TEXT)
    • Password (BDI_ID)
    • Confirm password (CPIN)
    • Nearest branch (BDB_LNK)
    • Category (BDC_LNK)

    See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.

    It is mandatory to set a Preset value when a field is added to the form but is hidden (Hidden checkbox is set). For checkbox fields, the Checked by default option acts as a preset value.

    Display in transit information on 'Reservations not yet available' grid

    Previously reservations in transit to the pickup location would not display this information, and would no longer display the reservation's place in queue. This could cause confusion for borrowers viewing their Reservations not yet available at the OPAC borrower dashboard.

    This new feature will now display Item in transit for reservations that have been allocated but are not yet available at the pickup location. For head of family borrowers, the text will also be displayed for Reservations not yet available for any dependent borrowers.

    The 'Item in transit' text uses the Danger alert text colour from the CSS Styles tab of the OPAC HTML Configurator.

     Custom welcome emails for borrower categories

    This feature requires license version 11.7 or later.  

    Previously, the same welcome email was sent to all borrowers regardless of category. Welcome email content can now be customised for borrower categories, with different email content for borrowers in a category based on whether they self-register online or are registered by staff.

    This feature was introduced after being requested by Kildare Libraries on the Spydus Ideas Portal (SPY-I-1244). This idea received 60 votes!

    A new Welcome email section has been added to the Notice Settings tab of Borrower Categories Maintenance, allowing staff to create personalised email content for each borrower category. See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.

    Additional customisation of OPAC login text

    Some new fields have been added to the Login section of the OPAC HTML Configurator to allow staff to customise more text labels in the OPAC login box.

    It is now possible to set the text labels for:

    • Log in text (highlighted yellow)
    • Information text (highlighted blue)
    • Library account text (highlighted green)
      • This text is displayed if both SSO & local login are enabled
    • Online registration prompt (highlighted pink)

    See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.

     Allow customisation of home, join and login button icons

    Staff can now customise the icons for the Home, Login, and Join buttons at the OPAC.

    • The Home icon is displayed on desktop and mobile
    • The Login and Join online icons will be displayed on mobile

    Updated error message for borrowers without a hoopla account  

    This feature requires a minimum patch or release version:  

    • 10.7.14,
    • 10.8.12,
    • 10.9.10,
    • 11.7

    As the hoopla eContent APIs do not provide the ability for Spydus to register new borrowers with hoopla, borrowers without a hoopla account must register at hoopla's website before being able to access harvested eResources from the Spydus OPAC.

    Previously when borrowers without an account attempted to borrow hoopla resources, Spydus was providing a generic error message that wasn't helpful to patrons or staff. For libraries on Spydus 10.7-10.9, a more descriptive error message is now displayed with a link to hoopla's site.

    For libraries on Spydus 11, a new When unable to create an account message in hoopla's section of the eContent Help tab in the OPAC HTML Configurator allows customisation of this error message.

    See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.

    Work Instance display enhancements

    A number of optional parameters have been introduced to allow library staff to customise the titles table at the Brief & Full Display of Work records at the OPAC.

    Customise Work Instance title display

    New parameters in the Work Instance section on the General tab of the OPAC HTML Configurator allow staff to control the display of titles linked to a work record. These changes were made based on feedback received that OPAC visitors found it unintuitive to have to click to expand the titles table.

    • Show titles on page load - If checked, the Titles table will be automatically expanded on page load
      • If unchecked, the Titles toggle link will be displayed to allow visitors to click to expand & collapse the titles table
    • Show toggle link - If Show titles on page load is checked, Show toggle link will be revealed. Checking Show toggle link will display a text link to allow visitors to click to expand & collapse the title panel (see the yellow highlighted Titles in the screenshot below)
      • If unchecked, the titles table will be fixed in the expanded state
    • Toggle link text - If Show toggle link is checked, Toggle link text will be revealed. The Toggle link text is the text of the link clicked to toggle the expanded/collapsed state of the titles table
    • Show titles table header - If checked, the titles table header row will be displayed (see the teal coloured row in the screenshot below)

      Update tab description and order at Work Instance display

    The Bibliographic record format tabs representing each in a Work record may now be renamed and re-ordered. The table containing the formats, their description and the order is in the Work Instance section on the General tab of the OPAC HTML Configurator.

    In the screenshot below:

    • the Book format has been renamed Paperback
    • the eAudiobook format has been renamed eListen
    • the eBook format has been renamed eRead

    Changes to the Bibliographic record format descriptions will apply to the Brief & Full Display of Work records, as well as the facet labels when refining on Record format.

    For more information on configuring the titles table & bibliographic record formats, see the Spydus Maintenance features in 2024R3 page.

    Show 'Previously borrowed' and 'New title' banners on Work records

    This feature was introduced after being requested by Jamie at Joondalup on the Spydus Ideas Portal (SPY-I-1406). This idea received 32 votes!

    Borrowers can now identify new and previously borrowed titles at the Brief and Full Display of Work records.

    Previously, borrowers couldn't identify a previously borrowed or new title unless clicking through to the bibliographic record.

    The Enable Previously Borrowed visual indicator and Enable New Title visual Indicator must be enabled in their respective sections on the General tab of the OPAC HTML Configurator.

    The text size, weight and colour, and the background colour of the visual indicators may be customised in the Alert & Indicators section of the CSS Styles tab of the OPAC HTML Configurator.

    Display alternate title (880 tag) on OPAC work display

    The OPAC now displays alternate titles for Work records in both Brief and Full Display. This functionality was previously supported only for Bibligraphic records.

     Hide locations from OPAC search lookups

    This feature was introduced after being requested by Ken from City of Melbourne on the Spydus Ideas Portal (SPY-I-1379). This idea received 29 votes!

    Previously Locations could be suppressed from OPAC display, meaning that items from those locations would not appear in OPAC searches. However the Location itself would still appear in search form lookups. A new parameter has been added to Locations Maintenance that allows Locations to be removed from these lookups.

    The affected search form are:

    • Library Catalogue (Bibliographic) Advanced Search
    • Library Catalogue (Work) Advanced Search
    • Equipment Booking Advanced Search
    • Article Advanced Search

    See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.

    Allow entire surface of flipped cards to be clickable

    This feature was introduced after being requested by Jamie at Joondalup on the Spydus Ideas Portal (SPY-I-1396). This idea received 14 votes!

    OPAC containers now support full card clicking for Masonry and Image list containers to enhance usability. Previously only the heading text was clickable, now the entire flipped card surface is hyperlinked in for single-click navigation to destination links. 

    Improve display of materials in OPAC booking table

    In previous versions, materials were only displayed together in the booking table if they had consistent:

    • minute/day type,
    • maximum adjacent slots/days,
    • opening times,
    • slot periods

    From 2024R3, materials will be displayed in the same table regardless of their opening locations and slot times, if they have the following properties aligned:

    • minute/day type,
    • maximum adjacent slots/days
    The slot periods at OPAC are defined at Maintenance > Circulation > Collection, under Booking slot in minutes / Booking slot of 1 day.
