    HTML Configurator - Item material type icons section
    In This Topic

    Spydus may be configured to show an icon based on Item material type instead of the Bibliographic Record type.

    This feature requires server version 10.5.1 or higher.

     To enable;

    1. Navigate to Maintenance > OPAC & Enquiry > HTML Configurator.
    2. Expand the Item material type icons section.
    3. Check the box to Enable use of item material type icons.
    4. Use the icon picker to select appropriate icons for the Material types.

    Ensure the following considerations are taken into account:

    If Material type is not set in the Item record/s, the Bibliographic record format will continue to display.

    If there are mixed Material types linked to a single bibliographic record, only one icon may display.

    The available Material types are drawn from the KCE table in Table Manager.

    Item material type search options

    If using Item material type, it may be desirable to enable search options and facets to allow refining and filtering based on Item material type.

    To enable Item material type search fields in the Advanced Search at the OPAC;

    1. Navigate to Maintenance > OPAC & Enquiry > HTML Configurator.
    2. Expand the Search section.
    3. Under the Advanced heading, check the Show item material type option.
      • Un-check the Show bibliographic material type option.
    4. Click Save or Save & Close.

    To enable the Item material type facets at search results in the OPAC;

    1. Navigate to Maintenance > OPAC & Enquiry > Enquiry Parameters.
    2. From the View menu, select the WPAC option.
    3. From the Format menu, select the BIBENQ option.
    4. Click the Facets tab.
    5. Scroll to the BIBMTYP facet option, select it and click Edit, then check the Enabled box.
      • Review the BTY, VMT and SRT material type facets, and configure as required.
    6. Click Save or Save & Close.
