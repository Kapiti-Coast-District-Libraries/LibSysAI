    Recommendations queries
    In This Topic

    In the My account section of the OPAC, borrowers can view recommendations for titles to loan, based on titles that they have previously loaned.

    Borrowers can also opt in to Recommendation alerts.

    images/HTML_CONFIGURATOR_MY_ACCOUNT_RECOMMENDATIONS_thumb.png

    Click image to enlarge

    To enable or disable the Recommendations containers, check the FULL_RFY Type in the BRW Display Parameters.

    What populates the Recommendations containers and alerts?

    The Recommendations query takes the following steps:

    1. Searches for all loans linked to the borrower's IRN,
    2. Translates those loans to bib records,
    3. Those bib records are then translated to the authority records linked to those bibs (author, subject, etc)
    4. Some tags are stripped from those records, and the authority records are translated back to bibliographic records

    The set of records the query has at this point are all bib records in Spydus with similar authority records as titles previously loaned by the borrower.

    The query then:

    • Removes bibs which the borrower has previously loaned, and;
      • for fiction recommendations, includes the fiction indicator and specific fiction codes
      • for nonfiction recommendations, excludes the above

    In an alert email, the system will then select the number titles specified by the borrower, with a preference for titles with newer holdings.
