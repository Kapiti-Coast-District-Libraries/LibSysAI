    Spydus OPAC features in 2024R2
    In This Topic

    OPAC timeout page display

    The OPAC HTML Configurator now allows staff to configure timeout messages that will display after a period of inactivity at the OPAC.

    Different messages may be displayed based on whether a visitor is logged in or not.

    images/2024R2_OPAC_CONFIG_TIMEOUT_EXAMPLE_thumb.png

    Click image to enlarge

    See Maintenance features in 2024R2 for more information.

     Optionally display announcements on all pages

    If configured in the OPAC HTML Configurator, announcements may now be displayed on all pages at the OPAC. Previously, announcements were only displayed at the home page.

    Customised failed login messages

    It is now possible to set the page title for the failed login page, as well as custom messages for when borrower login fails due to:

    • Entering the wrong borrower ID/username or password
    • Attempting to login with a borrower ID that has been suspended, cancelled, or has borrower traps or alerts applied that prevent OPAC login
    • Performing SSO login, but no matching Spydus account exists for that SSO ID

    See 2024R2 Maintenance new features for more information.

     Add support for multiple custom website searches

    In previous versions, it was only possible to create one 'Custom search' in the quickfind menu. This new feature allows staff to group multiple custom searches under a heading in the quickfind search dropdown, giving borrowers the ability to search other websites from the OPAC (searches will be launched at the target site in a new browser tab).

    For more information see 2024R2 Maintenance new features.

    Add Meta Pixel analytics to OPAC

    If enabled in the Analytics section in the General tab of the OPAC HTML Configurator, Meta (formerly Facebook) Pixel may be used to track site analytics.

    US245957 (11.2.1/2024R2) Customise options for the number of records displayed per page when searching

    It is now possible for staff to configure multiple options for the number of records to display when OPAC visitors perform searches.

    The default ('Selected') number of records per page will be displayed when performing searches from the quick search bar. Visitors may select any of the configured options when searching from the search forms.

    For more information see 2024R2 Maintenance new features.

    Display custom favicon

    If a custom favicon is uploaded to the OPAC HTML Configurator, the favicon will be displayed on the browser tab, and/or on the browser bookmark for the site.

    For more information see 2024R2 Maintenance new features.

     Additional OPAC footer

    If enabled, an Additional footer may be displayed below the footer menu options.

    images/2024R2_MAINTENANCE_OPAC_ENQ_ADD_FOOTER_thumb.png

    Click image to enlarge

    For more information see 2024R2 Maintenance new features.

    Customisation of OPAC request forms

    It is now possible to create and customise request placement forms for use at the OPAC. These forms can be linked to directly, or presented for borrowers to select from in the new customisable request form selection page.

    images/2024R2_DISCOVERY_REQSEL_thumb.png

    Click image to enlarge

    It is now also possible to disable the 'Place request' button from the results of an external (Z39.50) search at the OPAC.

    For more information on configuring these features, see 2024R2 Maintenance new features.

    Add custom facets to OPAC searches

    It is now possible for staff to add custom facets to all OPAC searches. Custom facets use boolean queries to refine searches performed by OPAC visitors.

    images/2024R2_DISCOVERY_CUSTOM_FACETS_thumb.png

    Click image to enlarge

    For more information on configuring this feature, see 2024R2 Maintenance new features. 

    Additional options for reservation placement in a consortium

    For libraries operating as part of a consortium that also allow reservation placement on global holdings, some additional parameters have been implemented to control the behaviour of these reservations at the OPAC. These can be set in the Reservations section on the General tab of the OPAC HTML Configurator.

    The parameters control:

    • the default behaviour of when a reservation will be placed on local or global holdings
    • text labels to describe which holdings will be reserved

    For more information on configuring this feature, see 2024R2 Maintenance new features.
