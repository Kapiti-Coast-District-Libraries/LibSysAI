    Issue, renew or return a box set
    In This Topic

    Issuing a box set

    To issue a box set:

    1. From the Module Menu select Circulation, and load the borrower details.
    2. Scan or type the accession number of the box set.

    If you don't have the accession number, select  "Search box set" from the dropdown list and then click this button. This opens the Box Sets Search page. You can then find the set you want and click Select Record to issue it.

    Box sets can only be issued via the Circulation module. Attempting to issue via self-service will trigger the BOXSETISSFDONLY alert, which is critical and cannot be overridden. 

    Issuing a box set will display the loan as a single line item in the display view. Staff can click on the item in the loan view to navigate to the individual items/materials in this box set.

    Click on Child Loans to see the items/materials in the box set.

     

    Issuing items separately

    If you try to issue an item/material from a box set as a separate item, the ITMBOXSETLNK alert will be displayed. You can then choose  to issue the item/material or permanently remove it from the box set.

     

    If the issue proceeds,  the item's original collection is used to determine the loan policy.

    When the item/material is returned, it will again adopt the collection, location and loan policy of the box set, and an alert displays to advise the item is part of a box set.

    Incomplete box sets can still be issued as a single loan transaction.

    Renewing a box set loan

    A box set loan is a 'parent' loan.

    All child loans (the linked items and materials) cannot be renewed separately.

    If any item or material within the box set needs to be renewed, the whole box set loan will have to be renewed.

    Items and materials that have been issued as part of a box set are not displayed individually in the Circulation transaction pane.

    The only exception is if the item/material was issued separately as an individual loan. These items/materials can then be renewed separately in OPAC.

    You cannot renew box set loans through OPAC, only through Circulation or Staff Enquiry. The only exception is if items from the box set have been issued individually. These are able to be renewed via OPAC according to usual renewal policies.

    Returning a box set

    Items and materials in a box set can be returned individually or together.

    A box set loan will only be complete when all items/materials that were originally issued are returned.

    If a box set item has been issued separate to the box set, the ITMBOXSETLNK alert will notify Circulation staff that it's part of a box set.
