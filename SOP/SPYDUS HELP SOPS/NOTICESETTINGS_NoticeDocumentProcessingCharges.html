    Notice document processing charges
    In This Topic
    Select Inherit from default to inherit the settings from the default borrower category. Clear this option if you want to add, edit or delete the settings for the selected borrower category.

    To specify the notice processing charge in the borrower category

    1. From the Module Menu select Maintenance, Circulation and then Borrower Categories.
    2. Select the category you want from the list.
    3. On the Notice Settings tab, in the Notice Document Processing Charge group:
      • Click Add to add a new charge. You can then select the Delivery method, Print format, Charge code and Charge amount. Only the charge codes with Use as notice processing charge set to Yes will be listed in the Charge code list.
      • Click Edit to edit an existing charge.
      • Click Delete, and then OK at the confirmation prompt to delete an existing charge.
    4. Click Save or Save & Close.

    Borrower category and notices policy

    You can specify the notice processing charge in the borrower category and/or in the notice policy. Here are a couple of reasons why it’s better to specify the charge in the borrower category rather than the notice policy.

    • The processing charge is for the “notice delivery document” (or NDD), i.e. the physical notice that gets sent to the borrower. Since one NDD can have many notices (NTCs), it makes more sense to specify the processing charge for the NDD (in the borrower category) rather than for each NTC (in the notice policy).
    • If the NDD has more than one NTC we look at the processing charge per notice type per level and apply whichever is the greater. This means that you can’t always tell which processing charge will be used. You’ll see examples of this below.

    Below are some examples to show how the processing charge for each NDD is calculated if you DO specify a processing charge in the notice policy. We've specified different amounts for each policy to try and make it easier to see which processing charge is used. In all these examples, the notice document processing charge in the borrower category is $1.20.

    Example 1

    In the first example the borrower has three overdue items all under the same notice policy.

    Notice Polices Proc. charge
    Overdue level 1 notice $0.75

    If all three overdue level 1 notices are on one NDD then this NDD will have two charges:

    $0.75 1 x Overdue level 1 notice charge.

    $1.20 1 x Notice document processing charge from the borrower category.

    Examples 2

    In the second example the borrower has one reservation available notice, two overdue level 1 notices and three overdue level 2 notices.

    Notice Polices Proc. charge
    Reservation available notice $2.65
    Overdue level 1 notice $0.75
    Overdue level 2 notice $1.45

    If all six notices are on one NDD then this NDD will have four charges:

    $2.65 1 x Reservation available notice charge.

    $0.75 1 x Overdue level 1 notice charge.

    $1.45 1 x Overdue level 2 notice charge.

    $1.20 1 x Notice document processing charge from the borrower category.

    Example 3

    In the third example the borrower has two overdue level 1 notices under one policy and two overdue level 1 notices under another policy. The processing charge for the policies is different.

    In the second example the borrower has one reservation available notice, two overdue level 1 notices and three overdue level 2 notices.

    Notice Polices Proc. charge
    FICTION Overdue level 1 notice $0.75
    REFERENCE Overdue level 1 notice $4.30

    If all four notices are on one NDD then this NDD will have two charges:

    $4.30 1 x Overdue level 1 notice charge. We use the processing charge for the REFERENCE notice policy because 4.30 is greater than 0.75.

    $1.20 1 x Notice document processing charge from the borrower category.
