    Consortium payments - allow payment/refund of charges from other institutions
    In This Topic

    To specify whether you can pay or refund charges incurred at other institutions:

    1. From the Module Menu select Maintenance, Circulation and then General Parameters.
      • Set Consortium payments - allow payment to Yes or No. If you set it to Yes then you can pay charges incurred at other institutions.
      • Set Consortium payments - allow refund to Yes or No. If you set this to Yes then you can refund charges incurred at other institutions.
    2. Click Save or Save & Close.
    Null means the same as No.
