    Birthday alert
    In This Topic
    Select the checkbox to the left of a parameter to inherit the setting from the default borrower category. Clear this checkbox if you want to change the setting for the selected borrower category. You can also click Inherit All and Inherit None to inherit or disinherit the settings for all parameters.

    You can specify that a birthday alert is displayed at the Front Desk when borrowers in a selected category reach a certain age. To do this:

    1. From the Module Menu select Maintenance, Circulation and then Borrower Categories.
    2. Select the category you want from the list.
    3. On the Details tab, set the following parameters.
      • Birthday alert age. Specify the age in years.
      • Birthday alert cutoff period. Specify the cutoff period in days. If you don’t specify a period then 60 days will be used by default.
      • Birthday alert ID. Select the alert to be displayed. You can use the BIRTHDAY alert or create your own alert if you wish.
    4. Click Save or Save & Close.

    For example, say the Birthday alert age is 10, the Birthday alert cutoff period is 3 days and the borrower’s birthday is 21 May 2006. The Birthday alert ID will be displayed the first time you scan the borrower’s barcode on 18, 19, 20, 21, 22, 23 or 24 May 2016.
