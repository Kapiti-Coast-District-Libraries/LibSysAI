    Charge parameters for Self Service
    In This Topic

    To specify whether a charge can be paid via Self Service:

    1. Navigate to Maintenance > Circulation > Charges.
    2. Select the desired charge from the list.
    3. Set Allow payment via Self Service to Yes or No.
    4. Click Save or Save & Close.
    Null means the same as No.  

    LCF and SIP2 charge type 

    The LCF charge type feature requires commissioning is server versions prior to 10.8.2.  

          If a charge is payable via self service, it may be desirable to set the charge type for the relevant payment interface. To set the charge type:

          1. Navigate to Maintenance > Circulation > Charges.
          2. Select the desired charge from the list.
          3. Select the desired LCF charge type from the dropdown list.
            • The available LCF charge types are determined by the CHT charge type in the LCF API standards
          4. Select the desired SIP2 charge type from the dropdown list.
            • The available SIP charge types are determined by the 3M Standard SIP2 fee types
          If a charge may be paid at self service, and the charge type has not been selected, the default 01 charge code (other/unknown) will be applied.
