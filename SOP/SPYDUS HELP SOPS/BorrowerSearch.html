    Search for borrowers in Circulation
    In This Topic

    Quickfind search

    For a Quickfind search simply type or scan information in the Quickfind box. The option in the Search box determines what you're searching for.

    • Borrower or item. Use this option to search by borrower name, borrower barcode or item barcode. The information can be entered by typing or scanning. This is the default search. Select Keyword to search for borrowers whose name contains the text entered or Browse to search for borrowers whose name begins with the text entered.
    • Borrower. Search for borrowers by borrower name or by typing or scanning the barcode.
    • Username. Search for borrowers by username.
    • Item. Search for items (keyword only) by title or by typing or scanning the barcode.
    • Bibliographic. Search for items by title (keyword only).

     

    If there is only one borrower with the entered name, the full details are displayed.

    If there is more than one borrower with the entered name, a list of borrowers is displayed. Click on the borrower name hyperlink to display the full details.

    If you have conducted a browse search, for smith,j for example, and there is no borrower of that name on the database, a list of borrowers starting with the next borrower alphabetically after smith,j  displays (this might be Smith, Ken , for example).

    Advanced search

    For a more advanced search:

    1. Leave the Quickfind box empty.
    2. Select the type of Search from the drop down menu.
    3. Click the magnifying glass to open the relevant advanced search form.

    Using the Borrower Search form

    To search for borrowers using the Borrower Search form:

    1. Click Search pages > Borrower to display the search page.
    2. Enter your search criteria and then click Submit.
    Enter Select To search for
    smith Browse All borrowers whose surname begins with "smith".
    smith Keyword All borrowers whose name contains "smith".
    smith,j Browse All borrowers whose surname begins with "smith" and whose first initial is "j".
    j smith Keyword All borrowers whose name contains the initial "j" and "smith"
    smith john Browse All borrowers whose surname begins with "smith" and whose first name is john".
    john smith Keyword All borrowers whose name contains "john" and "smith".
    You can select multiple choices within a multi-choice menu using SHIFT+click or CTRL+click, for example to select several branches. Use CTRL+click to deselect a selection. You can also click the “any” option to deselect all the selections within a multi-choice menu or click Clear to deselect all selections.
    1. If more than one borrower is found, then these borrowers will be listed alphabetically. Use Sort Set to sort by name, barcode, branch/name and category/name, registration expiry date and birth date, and Refine Set to add additional search criteria and reduce the number of borrowers found. Facets also allow staff to refine the results.

    Borrower messages and alerts

    If a message or an alert pops up when you select the borrower then you can only continue by pressing the Esc key, pressing the O key, or clicking OK.
