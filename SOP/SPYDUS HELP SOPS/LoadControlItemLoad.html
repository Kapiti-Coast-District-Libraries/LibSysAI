    Add an item load control
    In This Topic

    To add an "item load" load control:

    1. From the Module Menu select Cataloguing, MARC and then Load Control Maintenance.
    2. Click New.
    3. Specify the new ID, select the "bibliographic" record Type and click OK.
    4. Specify a description.
    5. Select "Item Load" as the load type.
    6. On the General tab, select the matching tags, tags not wanted and tags wanted just as you would for a normal MARC record load.
    7. Set up the parameters on the Item Load tab.
    8. Click OK, and then click Save or Save & Close.

    Specify item tag

    Click Specify item tag to specify which tag stores the item information. This is typically the 984 tag, but it can be any 9XX tag. You can only specify one tag.

    Specify the subfields you'll be using. For example you might specify #b for Location, #c for Collection and #d for Accession number. Only the subfields for Location and Collection are mandatory.

    We don't load the call number. Instead we load the subfields that make up the call number — Prefix, Classification scheme and Suffix.

    Replace existing item records

    When items are loaded we match on accession number and RFID (if there is one). Select Replace existing item records if you want matching items to be replaced, i.e., if the incoming item matches an existing one then the existing one will be replaced.

    Loading non-physical items

    On the Item Load tab you can specify the subfield for the Item type.

    If you set this subfield to “N” in the record then the item will be loaded as a non-physical item. If you set this subfield to anything other than "N" or leave it null then the item will be loaded as a physical item.

    Typically you might set the subfield to "I" to make it obvious that it's a physical item.
