    Processing archival requests
    In This Topic

    Edit an archival request

    You can edit details of an archival request to change the pickup date, pickup location, status, add a status note, assign the request to a staff member and/or add a request note.

    In Archives:

    1. From the Module Menu select Archives, Search Pages and then Archival Request and find the request you want.
    2. Click Edit at the Full Display.
    3. Make your changes and click OK.

    In Enquiry:

    1. From the Module Menu select Enquiry, Circulation and then Borrower and find the borrower who placed the request.
    2. Click Process Archival Requests at the Full Display. This will display the borrower's current archival requests.
    3. Select the request you want and click Edit.
    4. Make your changes and click OK.
    Rather than click Process Archival Requests at step 2 you could click the link to the borrower's current archival requests. Select the request you want, click Edit at the Full Display, make your changes, and click OK.

    In Circulation:

    1. From the Module Menu select Circulation, and load the borrower details.
    2. Click the link to the borrower's current archival requests.
    3. Select the request you want and click Edit.
    4. Make your changes and click OK.

    Move an item to the pickup location

    While an archival request item is on the move to the pickup location you can mark its movement status as "intransit" if you wish. Here are some ways you could do that.

    In Archives:

    1. From the Module Menu select Archives, Search Pages and then Archival Request and find the request you want. For example you might search for pending requests.
    2. Click Move at the Full Display.
    3. Change the Movement status to "Intransit", and click Move.

    In Enquiry:

    1. From the Module Menu select Enquiry, Circulation and then Borrower and find the borrower who placed the request.
    2. Click Process Archival Requests at the Full Display. This will display the borrower's current archival requests.
    3. Select the request you want and click Move.
    4. Change the Movement status to "Intransit", and click Move.

    Rather than click Process Archival Requests at step 2 you could click the link to the borrower's current archival requests. Select the request you want, click Move at the Full Display, change the Movement status to "Intransit", and click Move.

    In Circulation:

    1. From the Module Menu select Circulation, and load the borrower details.
    2. Click the link to the borrower's current archival requests.
    3. Select the request you want and click Move.
    4. Change the Movement status to "Intransit", and click Move.

    Receive an item at the pickup location

    Once the archival request item gets to the pickup location you can mark it as received for pickup if you wish.

    1. Find the item you want to receive. For example:
    • From the Module Menu select Archives, Search Pages and then Archival Request and find the request you want. Click Process at the Full Display.
    • From the Module Menu select Enquiry, Circulation and then Borrower and find the borrower who placed the request. Click Process Archival Requests at the Full Display.
    • From the Module Menu select Circulation, and load the borrower details. Click the link to the borrower's current archival requests.
    1. Select the request you want and click Receive. to change the movement status to "received for pickup".

    Issue an item

    You can issue an archival request item without using Move to mark it as "intransit" or Receive to mark it as "received for pickup".

    To issue an archival request item to a borrower:

    1. Find the item you want to issue. For example:
    • From the Module Menu select Archives, Search Pages and then Archival Request and find the request you want. Click Process at the Full Display.
    • From the Module Menu select Enquiry, Circulation and then Borrower and find the borrower who placed the request. Click Process Archival Requests at the Full Display.
    • From the Module Menu select Circulation, and load the borrower details. Click the link to the borrower's current archival requests.
    1. Select the request you want and click Issue.

    Return an item

    To return an item from a borrower:

    1. Find the item you want to return. For example:
    • From the Module Menu select Archives, Search Pages and then Archival Request and find the request you want. Click Process at the Full Display.
    • From the Module Menu select Enquiry, Circulation and then Borrower and find the borrower who placed the request. Click Process Archival Requests at the Full Display.
    • From the Module Menu select Circulation, and load the borrower details. Click the link to the borrower's current archival requests.
    1. Select the request you want and click Return.

    You'll see an intransit message if the item needs to return to its storage location. See Movement for Archives for details on how to receive an item at its storage location.
