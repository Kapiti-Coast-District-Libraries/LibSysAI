    OPAC details for Locations
    In This Topic

    Suppress items from OPAC display

    You can specify whether items are suppressed from OPAC in three places:

    • At the item level
    • At the location level
    • At the collection level
    Records may also be suppressed using an item status or at the bibliographic level. For more detailed information, see the Hierarchy of OPAC suppression article.

    Suppress particular items from OPAC

    At the item level you can specify whether a particular item is suppressed from OPAC. 

    1. Use Enquiry to search for and find the item you want, and then click Edit Holdings to open Item Maintenance.
    2. Select the item you want from the list.
    3. Click the Details tab.
    4. Click Filters.    
    5. Scroll to the General section and set the Override OPAC filter option as desired.
      • Display in OPAC
      • Suppress from OPAC
    6. Click OK, and then click Save or Save & Close.

    If you don’t set Suppress access from at the item level then Spydus will check the setting at the location level.

    Suppress items at a particular location from OPAC

     At the location level you can specify whether items at a particular location are suppressed from OPAC.

    1. From the Module Menu select Maintenance, Circulation and then Locations.
    2. Select the location you want from the list.
    3. Click the OPAC tab and review the OPAC Filter section.
    4. You can click Inherit from parent to inherit the settings from the parent location. Clear Inherit from parent if you want to change the settings for the selected location.
    5. Set the Override OPAC filter option as desired:
      • Display in OPAC
      • Suppress from OPAC
    6. Click Save or Save & Close.

    If you don’t set Override OPAC filter at the location level then Spydus will check the setting at the Collection level.

    Suppress items at a particular collection from OPAC

    At the collection level you can specify whether items at a particular collection are suppressed from OPAC.

    1. From the Module Menu select Maintenance, Circulation and then Collections.
    2. Select the collection you want from the list.
    3. Click the Filters tab and review the OPAC Filter section.
    4. You can click Inherit from parent to inherit the settings from the parent collection. Clear Inherit from parent if you want to change the settings for the selected collection.
    5. Set the Override OPAC filter option as desired:
      • Display in OPAC
      • Suppress from OPAC
    6. Click Save or Save & Close. 

    Location notes

    This section includes a rich text editor to add information regarding the Location, which will be visible at the Full Display of the Location record at the OPAC only. These notes will be displayed under an Additional Information heading, which may be enabled or disabled via Display Parameters.
