    Add, edit or delete items to spine label & barcode printing queue
    In This Topic

    To view the spine label requests:

    1. From the Module Menu select Cataloguing, MARC and then Spine Label & Barcode Printing.
    2. This displays the Details dialog. By default, the users, locations and/or collections that were selected the last time the requests were viewed will be selected now. But you can select the specific users and/or locations and/or collections you want. Click OK.
    3. You can then add, edit or delete requests.

    You can click Tools and then Options to see the requests for other users, locations and/or collections.

    Add spine label requests

    To add a single accession number:

    1. Click Add Spine, and then Manual Input.
    2. Select another user from the Add to user list if you wish.
    3. Scan or type the accession number, and then click OK to add the request to the list.

    To add several accession numbers:

    1. Click Add Spine, and then Scan Input.
    2. Select another user from the Add to user list if you wish.
    3. Scan or type the accession numbers.
    4. If you want to delete an accession number, click to select it and then click Remove.
    5. Click OK to add these requests to the list.

    Edit spine label requests

    To change the number of copies to be printed, the class number, the prefix or the suffix for a spine label, double click the request you want. Make your changes and then click OK to save.

    Delete spine label requests

    To delete spine label requests:

    1. Click to highlight the row you want to delete, and then click Delete.

    You can use SHIFT+Click to select consecutive rows, or CTRL+Click to select specific rows to delete.
