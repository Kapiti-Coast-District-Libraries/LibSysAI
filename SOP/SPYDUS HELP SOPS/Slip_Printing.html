    Slip printing
    In This Topic
    You can set up printed slips that are specific to a location. Contact Civica Support Staff for more information.

    Transactional slips

    "Transactional" slips are the ones that are printed automatically when you carry out a transaction, e.g. when you issue a loan or when a borrower pays a charge.

    Receipt slip This gets printed when you pay or waive a charge.
    Cash drawer receipt slip This gets printed when you close a cash drawer.
    Refund slip This gets printed when you refund a charge.
    Item status slip This gets printed when you return an item that has a status with Print item status slip set to Yes.
    Item note slip This gets printed when you return an item that has a note.
    Issue slip This gets printed when you issue or renew loans.
    Issue offline slip This gets printed when you issue or renew loans to a borrower when you're offline.
    Intransit slip This gets printed when you return an item at a location that's not its home location.
    Reservation slip This gets printed when you return an item and it's allocated for pickup at the current location.
    Reservation intransit slip This gets printed when you return an item and it's allocated for pickup at another location.
    Postal delivery slip This gets printed when you return an item and it's allocated for delivery to the borrower.
    Return slip This gets printed when return an item.
    The slip printing parameters control which transactional slips are printed automatically.

    On demand slips

    "On demand" slips are the ones you print when you want, e.g. click the Print Onloan Slip button on the Action Menu to print a slip listing a borrower's current loans.

    Outstanding (current) loans slip The Print Onloan Slip button is displayed on the Action Menu when you're viewing the borrower's loans at the Front Desk.
    Outstanding charges slip The Print Outstanding Charges Slip button is displayed on the Action Menu when you're viewing the borrower's charges at the Front Desk.
    Outstanding transactions slip The Print Outstanding Transactions Slip button is displayed on the Action Menu when you're viewing the borrower's transaction summary at the Front Desk. This slip shows onloan items, claimed returned items, claimed never borrowed items, reported lost items, lost item, reserved items available for pickup, reservations not yet available. standing reservations, charges, article requests, ILL article requests, ILL requests, purchase requests, research requests and undefined requests.
    Outstanding (current) reservation slip The Print Reservation Slip button is displayed on the Action Menu when you're viewing the borrower's reservations at the Front Desk. This slip shows reserved items available for pickup, reservations not yet available and standing reservations.
    Reservation Available slip The Print Reservation Avail Slip button is displayed on the Action Menu when you're viewing the borrower's reservations at Front Desk and the borrower has any allocated items at the pickup location.
    Overdue slip The Print Overdue Slip button is displayed on the Action Menu when you view a borrower's overdue loans at the Front Desk.
    Lost loans slip The Print Lost Loans Slip button is displayed on the Action Menu when you view a borrower's lost loans at the Front Desk.
    Reported lost loans slips The Print Reported Lost Loans Slip button is displayed on the Action Menu when you view a borrower's reported lost loans at the Front Desk.
    Claimed returned/never borrowed slips The Print Claimed Returned Slip and Print Claimed Never Borrowed Slip buttons are displayed on the Action Menu at the Front Desk when you view the items a borrower claims to have returned or never borrowed.
    Routing list The Print List button is displayed on the Action Menu when you edit a routing list.
    Profile matching slips Print Slips is displayed on the Action Menu when you're matching titles for borrowers on a round. This will print a slip for all the matched titles on the selected round. You can then use these slips to find the items on the shelf. The slips are printed in shelf order, i.e. by call number, to make it easier to find the item on the shelf.
