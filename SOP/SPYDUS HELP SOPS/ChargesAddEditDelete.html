    Add, edit or delete charges for a borrower
    In This Topic

    Add a new charge

    To add a new charge:

    1. Display the Charges dialog.
    2. Click Add Charges.
    3. Select the charge you want in the Description, specify the amount, add a note, etc.
    4. Click Add to add this charge.
    In the Description you'll only see the charges that are active for the borrower's category (i.e. those listed on the Charges tab in the borrower category) and that have Allow manual setting set to yes in the charge code parameters.

    Edit the notes for a charge

    To edit the notes for a charge:

    1. Display the Charges dialog.
    2. Select the charge you want.
    3. Click Edit, make your change and click Update.

    Delete charges

    To delete charges:

    1. Display the Charges dialog.
    2. Select the charge or charges you want to delete.
    3. Click Delete, and then click Yes at the confirmation prompt.

    Add a deposit

    Set Deposits active to Yes in the Circulation General Parameters to enable the Add Deposit option.

    Borrowers can choose to add a deposit on their account so that payment can be automatically made from their account when required. Whether a charge is paid automatically is specified in the Deposit level for each charge set up in the borrower category.

    To add a deposit:

    1. Display the Charges dialog.
    2. Click the down arrow next to Add Charges, and then click Add Deposit.
    3. Select the type of deposit you want in the Description, specify the amount and click Add.
    4. Select the payment mode, and click OK.

    Add a refund

    To add a refund:

    1. Display the Charges dialog.
    2. Click the down arrow next to Add Charges, and then click Add Refund.
    3. Select the type of refund in the Description, specify the amount, add a refund note if you wish, and click Add.

    If the borrower has money on account and no outstanding charges then you'll see the option to Close account on the Add Refund dialog. Select this option and the amount is automatically set to the value of the money on account. You can then click Add to refund this amount.

    The refund notes are set up in the RFN lookup table, and can be maintained using the Table Manager.

    Link a refund to a previous charge

    You can link a refund to a previous charge if you wish. To do this:

    1. Display the Charges dialog.
    2. Click the down arrow next to Add Charges, and then click Add Refund.
    3. Select the type of refund in the Description.
    4. Click the Previous charge button. This opens an in-app browser showing a list of the borrower's previous charges.
    5. Select the particular charge you want, and click Select Record at the Full Display.
    6. Add a refund note if you wish, and click Add.

    Rather than use something like "General refund" as the refund type, you might want to add a new type and use this, e.g. "Refund previous charge".

    Add an ad hoc charge for a borrower

    To add an ad hoc charge for a borrower:

    1. Display the Charges dialog.
    2. Click the down arrow next to Add Charges, and then click Add Ad Hoc Charges.
    3. Double-click a charge in the Available charges to add it to the Applied charges list.

    Once a charge is added to the Applied charges list, you can click the cursor in the Amount column and/or the Qty column to change the value.

    You can click Add and then Add Refund to add a refund to the Applied charges list.
