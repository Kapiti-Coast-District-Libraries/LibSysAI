    Upload records in a MARC file
    In This Topic
    Before you upload the records for real, you might wish to run a test load.

    Once you have opened a file of MARC records, you can upload them to the database:

    1. Select the record or records you want to load.
    2. Click Upload.
    3. Select the load control you want to use.

    The Use interactive load (single record load only) checkbox is available if you are loading one record rather than multiple records. If you select this option then you'll be prompted whether to select a matching record or create a new one.

    The Load as part of previous batch checkbox is only available if you are loading Weblink bibliographic records.

    The Don’t prompt before loading checkbox should usually be selected. This means you will not be prompted to select the load control every time you upload a record. If you do want to be prompted every time then clear this checkbox.

    1. Click Select.

    Click Do not stop on errors if you want to continue loading records even if you encounter errors, i.e. you want the load to continue and you will go back later and take a look at the record or records that have errors.

    1. Click Start Load.

    If you did not select the Don’t stop on errors option and you encounter an error then you will see an error dialog. You can then:

    • Click Stop if you want to stop the load.
    • Click Skip if you want to skip the current error and continue with the load.
    • Click Don’t stop on errors and then click Skip if you want to skip this error and any others you may encounter, and continue with the load.
    1. Click Finish Load when the load is completed.
    When a MARC load is completed, a message summarising the outcome will be sent to the Messages inbox of the user who initiated the load. Requires Spydus server version 10.2 or higher.
