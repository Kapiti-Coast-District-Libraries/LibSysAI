    Item status
    In This Topic

    What would you like to know about?

    Allow bookings Allows items with this status to be booked.
    Allow deletion Allows items with the selected status to be deleted.
    Allow issues Allows items with the selected status to be issued.
    Allow manual removal Allows the status to be removed manually using the Change Item Status command.
    Allow manual setting Allows the selected status to be set manually using the Change Item Status command.
    Allow reservations Allows items with the selected status to be reserved.
    Are items available at reservation Set this to Yes and items with the selected status will be considered as available when you try to reserve them.
    Block allocation Blocks allocation of reserved items with the selected status.
    Block change to location or collection Set this to Yes and items with the selected status cannot have their location or collection changed.
    Block profile matching Set this to Yes and items with the selected status will not be used in profile matching.
    Block recalls Set this to Yes and items with the selected status cannot be recalled.
    Block returns Set this to Yes and items with the selected status cannot be returned.
    Cancel bookings automatically Cancels bookings automatically when this status is set.
    Cancel reservations automatically Cancels unallocated reservations automatically when the selected status is set.
    Display alert at issue Displays an alert when an item with the selected status is issued.
    Display alert at return Displays an alert when an item with the selected status is returned.
    Exclude from loan quota validation Set this to Yes if loans with the selected loan status are to be excluded from loan quota validation at issue. 'Yes if lost charge paid' setting is only applicable for lost status.
    Exclude from on loan slip Set this to Yes if loans with the selected loan status are to be excluded from the on loan slip.
    Extend hold period Extends the hold period for reserved items and prevents an allocation notice from being sent. This would typically be set to Yes for the "intransit" status.
    Freeze fines when status set Freezes fines when the selected status is set.
    LCF circulation status code (CIS) Circulation status code (CIS) applied to an item entity in LCF API. If not set and no matching CIS code found then Available (03) status will be used.
    Period when status automatically removed Specifies the number of hours after the status has been applied when it should be automatically removed.
    Interrelationships Defines the item statuses that can be set or removed depending on which statuses are already set.
    OPAC description The item status description to be displayed in OPAC.
    Print item status slip Prints an item status slip when an item with the selected status is returned.
    Remove status at issue Removes the status automatically when an item is issued.
    Remove status at return Removes the status automatically when an item is returned.
    Routing status Status may be set in routing list maintenance.
    Suppress from OPAC display Set this to Yes if items with the selected status are to be suppressed from OPAC display.
    Trap when offline Set this to Yes if items with the selected status are to be trapped when you're using Circulation offline.
