    Join online section

    The Join online / Update personal details section contains a number of parameters utilised in online borrower registration, and the updating of borrower details at the OPAC.

    To configure these parameters;

    1. Navigate to Maintenance > OPAC & Enquiry > OPAC HTML Configurator.
    2. Scroll to the Join online / Update personal details section and click the header to expand.
    3. Configure parameters as required.
    4. Click Save or Save & Close.

    For the configuration of online registration forms, including fields, borrower category, minimum age, etc., see Borrower Join Online.

    General

    Enable Addressify

    Addressify is an Australian address auto-complete verification validation geocoding and parsing web service.

    If Addressify is enabled here, ensure that the API key is set in Third-Party Settings.

    Circulation Borrower Registration

    If Addressify is enabled and an API key supplied, the Full address subfield in Borrower Registration Parameters will be displayed in the Circulation borrower registration screen to allow address validation and autocomplete.

    OPAC Borrower Registration

    If Addressify is enabled and an API key supplied, ensure that the address autocomplete fields are added to the online registration forms in Borrower Join Online.
    For residential address, use Address [BDA_AUTOCOMPLETE].

    For mailing address, use Address [BDM_AUTOCOMPLETE].

    Enable manual entry of address

    If enabled, this option will enable manual entry of address details when Addressify is enabled in the OPAC borrower registration form. This may be required for new or unmapped addresses not present in Addressify's database.

    Enable Postcode Autocompleter

    This feature is currently only supported in the United Kingdom & Singapore. 

    If selected, this option will enable the use of getting a list of addresses from the postcode entered (based on data loaded into the EDATA table; this data is only loaded in the relevant countries).

    Join online selection

    The Join online selection section allows staff to configure the display of the selection form that is displayed when there are at least two online registration forms enabled. This page will be displayed when an OPAC visitor clicks the join online link to allow the borrower to select the most appropriate registration form. If there is only one online registration form enabled, the selection page will not be displayed and visitors will be sent directly to the enabled form.
