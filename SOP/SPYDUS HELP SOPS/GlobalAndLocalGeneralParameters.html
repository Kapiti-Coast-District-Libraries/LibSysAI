    When are global general parameters used rather than local ones
    In This Topic

    Several reservation parameters can be set in both the (local) general parameters in the global general parameters.

    • Allow OPAC reservations if on order.
    • Allow staff reservations if on order.
    • Check availability for staff.
    • Check availability for OPAC users.
    • Check availability at pickup location (staff).
    • Check availability at pickup location (OPAC).
    • Allow staff reservations if no holdings.

    This topic explains when we use the local ones and when we use the global ones.

    Allow OPAC/staff reservations if on order

    When determining whether to show the Place Reservation option in enquiry/OPAC.

    • We use the local parameters:
    • In local enquiry/OPAC if Global reservations active is No.
    • We use the global parameters:
    • In local enquiry/OPAC if Global reservations active is Yes.
    • In global OPAC.

    When populating the Items to reserve dropdown on the reservation placement page.

    • We always use the local parameters since this applies only in local enquiry/OPAC.

    When populating the Institution dropdown when Global reservations - select institution first is set to Yes. 

    • We always use the local parameters for each institution.

    When validating holdings on a global level reservation ("Any item from any institution") placed in local enquiry/OPAC.

    • We always use the local parameters for the pickup branch's institution.

    When checking the number of copies requested.

    • We use the local parameters:
    • For a local level reservation, i.e. "Any item from any location".
    • We use the global parameters:
    • For a global reservation, i.e. "Any item from any institution".

    When checking the availability.

    • We use the local parameters:
    • For a local level reservation, i.e. "Any item from any location".
    • We use the global parameters:
    • For a global level reservation, i.e. "Any item from any institution".

    Check availability for OPAC users/staff

    When determining whether to show the Place Reservation option in enquiry/OPAC.

    • We use the local parameters:
    • In local enquiry/OPAC if Global reservations active is No.
    • In local enquiry/OPAC if Global reservations active is Yes and there are no reservable holdings at another institution.
    • We use the global parameters:
    • In global OPAC.
    • In local enquiry/OPAC if Global reservations active is Yes and there are reservable holdings at another institution.

    When validating a reservation placement.

    • We use the local parameters:
    • For a local level reservation, i.e. "Any item from any location".
    • We use the global parameters:
    • For a global level reservation, i.e. "Any item from any institution".

    Check availability at pickup location OPAC/staff

    When validating a reservation placement.

    • We use the local parameters:
    • For a local level reservation, i.e. "Any item from any location".
    • We use the global parameters:
    • For a global level reservation, i.e. "Any item from any institution".

    Allow staff reservations if no holdings

    When determining whether to show the Place Reservation option in enquiry/OPAC.

    • We use the local parameters:
    • In local enquiry/OPAC if Global reservations active is No.
    • We use the global parameters:
    • In local enquiry/OPAC if Global reservations active is Yes.
    • In global OPAC.
