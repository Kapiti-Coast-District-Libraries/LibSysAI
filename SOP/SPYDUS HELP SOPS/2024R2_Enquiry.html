    Spydus Enquiry features in 2024R2
    In This Topic

     Additional sorting options available in Enquiry module

    Additional sorting options have been added to loan, reservation and charge enquiry results. These new options are available in the Sort by dropdown menu at the bottom of the relevant search forms, or by clicking the Sort Set button at the Brief Display of search set.

    New sorting options:

    Loan

    • Due Date (descending)
    • Issue Date (descending)
    • Title
    • Title (descending)
    • Author
    • Author (descending)

    Reservations

    • Date Placed (descending)
    • Expiry Date (descending)
    • Complete Date (descending)
    • Date Allocated (descending)
    • Hold Date (descending)
    • Title
    • Title (descending)

    Charges

    • Code
    • Code (descending)
    • Title
    • Title (descending)

    Add custom facets to Enquiry searches

    It is now possible for staff to add custom facets to all Enquiry searches. Custom facets use boolean queries to refine searches performed by staff.  

    For more information on configuring this feature, see 2024R2 Maintenance new features. 

    Option to display an additional section on search forms

    The new Search Forms feature in the Maintenance module allows staff to add a rich-text section (including images, HTML, etc) at the bottom of any Enquiry search form.

    Additional options for reservation placement in a consortium

    For libraries operating as part of a consortium that also allow reservation placement on global holdings, some additional parameters have been implemented to control the behaviour of these reservations at the OPAC. These can be set in the Reservations section on the General tab of the Enquiry HTML Configurator.

    The parameters control:

    • the default behaviour of when a reservation will be placed on local or global holdings
    • text labels to describe which holdings will be reserved

    For more information on configuring this feature, see 2024R2 Maintenance new features.
