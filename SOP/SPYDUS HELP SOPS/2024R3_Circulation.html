     Material Maintenance enhancements

    Spydus 2024R3 includes an updated Material Maintenance interface and a new feature allowing an image to be assigned to material records, making it visible in OPAC as well as in Enquiry.

    Material Maintenance interface enhancements

    A number of changes have been made to the Material Maintenance interface in the Circulation module:

    • The Other tab has been added, where staff can assign an image to the material record (see more below)
      • the Notes and Date acquired fields have been moves to this tab
    • A Generate button has been added to the Barcode field to generate barcodes
    • The Sublocation field has been added, allowing staff to add a sub-location for the material
    • The No booking minutes field has been removed
    • The Details tab has been renamed the General tab

    Adding an image to a Material

    To assign an image to a Material record:

    1. Navigate to Circulation > Item > New Material to create a new material
      • To locate an existing material to add an image, navigate to Circulation > Search Pages > Materials and perform a search and select the Edit Material option
    2. Switch to Other tab
    3. In the Images section, upload an image file in .jpg, .png, .bmp, or other supported format.
    4. On the General tab, enter mandatory Material details
      • Description
      • Location (and Sublocation if required)
      • Collection
    5. Enter any other desired optional details   
    6. Click Save or Save & Close

    Once the material details are saved, the uploaded image will be displayed as the primary image for the material at the Brief & Full Display of the record at the OPAC and in the Enquiry module.

    Add 'Author' field to Circulation grid column options

    Spydus 2024R3 has added a new optional Author field to the column options in the Issues, Returns, and Loans grids. This field will be available when a user clicks on the ellipsis (...) in the column header and by default, the Author option is unchecked.

    This option is available at the following grids in Circulation:   

    • Returns
    • Issues
    • Loan displays (e.g. Current, Overdue, Lost)

    When the user checks the box to add the Author column, the column will be added beside the Title column (once added the column may be arranged by clicking and dragging the header).

    Additionally, two column headers have also been renamed:

    • Name is now Borrower Name
    • Borrower Number is now Borrower No.

    Add additional fields to Circulation borrower details panel

    This feature was introduced after being requested by Birmingham City on the Spydus Ideas Portal (SPY-I-1278). This idea received 43 votes!

    A large number of fields have been made available for display in the Circulation borrower details panel. These include:

    • Receive Marketing flag
    • Check Loan History flag
    • Slip Printing & Notice Delivery preferences

    Staff can enable or disable these fields for the borrower details panel in Maintenance > OPAC & Enquiry > Display Parameters by selecting:

    1. the Staff (CAT) View,
    2. the Borrower (BRW) Format,
    3. the CCDET (FULL_CCDET) Type
    All new fields are disabled in Display Parameters by default.

    Work level loan history validation

    This feature was introduced after being requested by Albany Public Library on the Spydus Ideas Portal (SPY-I-1401). This idea received 33 votes!

    Previously when using the Check loan history functionality in Spydus, the system checked loan history based on the IRN of the bibliographic record. This would mean that after loaning one instance of a work (e.g. an audiobook), issuing a different instance of the same work (e.g. a paperback) would not trigger the loan history alert (LOANHIST).

    A new Circulation General Parameter has been introduced to optionally expand loan history validation to the work level, where all instances of the work that have been previously loaned by the borrower will be included when performing the check.

    See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.

     Custom welcome emails for borrower categories

    This feature requires license version 11.7 or later.  

    Previously, the same welcome email was sent to all borrowers regardless of category. Welcome email content can now be customised for borrower categories, with different email content for borrowers in a category based on whether they self-register online or are registered by staff.

    This feature was introduced after being requested by Kildare Libraries on the Spydus Ideas Portal (SPY-I-1244). This idea received 60 votes!

    A new Welcome email section has been added to the Notice Settings tab of Borrower Categories Maintenance, allowing staff to create personalised email content for each borrower category. See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.
