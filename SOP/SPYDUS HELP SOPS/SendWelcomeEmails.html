    Send welcome emails and subscribe to newsletter
    In This Topic
    Select the checkbox to the left of a parameter to inherit the setting from the parent location. Clear this checkbox if you want to change the setting for the selected location. You can also click Inherit All and Inherit None to inherit or disinherit the settings for all parameters.

    To specify whether to send a welcome email when a new borrower is registered:

    1. From the Module Menu select Maintenance, Circulation and then Locations.
    2. Select the location you want from the list.
    3. On the Properties tab, set up the Welcome email parameters you want (see below for details).
    4. Click Save or Save & Close.

    Send welcome emails

    This specifies whether you're prompted to send a welcome email to new borrowers who are registered by staff.

    Null means the same as No.  

    Send welcome emails when joining online

    This specifies whether to send welcome emails to new borrowers who register online.

    Null means the same as No.  

    Cc address for welcome emails

    Welcome emails you send to new borrowers will be cc'd to this address. Spydus will automatically supply the Internet protocol so don't include it in the email address, for example, enter library@civica.vic.edu.au rather than mailto:library@civica.vic.edu.au.

    Cc address for newsletter subscription

    E-mails to subscribe or unsubscribe a borrower to/from the newsletter will be cc'd to this address. Spydus will automatically supply the Internet protocol so don't include it in the email address, for example, enter library@civica.vic.edu.au rather than mailto:library@civica.vic.edu.au.

    If you’ve specified the Newsletter subscription service URL in the Circulation General Parameters then we ignore the Cc address for newsletter subscription.

    Path to welcome email attachments

    Specifies the path to a file attachment for welcome emails. Any file type, for example .pdf or .doc, will be sent as an attachment.

    The file path specified must be accessible to the application server. This means that the file must not be stored on the local PC. You can include logos in the body of welcome emails. The logo needs to be provided to Spydus Support Staff for commissioning, in the correct size and format (usually a jpg the same size as would go on a letter head).

    Path to welcome email attachments when joining online

    Specifies the path to a file attachment for welcome emails to borrower who register online. If this isn't set then the Path to welcome email attachments is used.

    Examples

    Here are two examples of how the parameter settings will work.

    In this first example the welcome email is cc'd to the Cc address for welcome emails and to the Cc address for newsletter subscription.

    1. Register a new borrower.
    2. Click Yes at the prompt to send a welcome email.
    3. Click Yes at the prompt to subscribe to the library newsletter.

    In this second example the welcome email is cc'd to the Cc address for welcome emails. No email is cc'd to the Cc address for newsletter subscription.

    1. Register a new borrower.
    2. Click Yes at the prompt to send a welcome email.
    3. Click No at the prompt to subscribe to the library newsletter.
