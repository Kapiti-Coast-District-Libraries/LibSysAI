    Serial Issue Search
    In This Topic

    A Serial Issue Search will allow you to find a Serial Issue using the Issue Reference, Type, Bibliographic Title, Issue Title and/or Publication Date.

    Performing a Serial Issue Search will allow you to not only search for serial issues, but also to also create savedlists, further refine the search results and to translate results to other linked Spydus records (serial records, loans, requests, etc), and other translations that may be associated with the record. An example of how you might use the translate function when searching for Serial Issue records is to translate your results into all items related to your search.

    Translating a set of Serial Issues into the items associated with them will allow you to see at a glance where issues that a borrower is interested in lending are located, and if those items are available or not.
