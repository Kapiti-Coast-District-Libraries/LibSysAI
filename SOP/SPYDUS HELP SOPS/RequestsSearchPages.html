    Requests search pages
    In This Topic

    You can use the options under Requests > Search Pages or those under Enquiry > ILL/Requests to find inter-library loans (ILL), requests, item requests, article requests and requests for information and then edit those records if you wish.

     

    Quickfind

    You can use the Quickfind search to find request numbers, Bibliographic Info or Borrower, for example:

     

     Select the option you want from the Search dropdown, and then type your search terms. All searches will display the request number but you can base the search on the following categories below.

    • "Request Number" searches for the requests and the details.
    • "Bibliographic Info" searches for the Bibliographic records that are used in Requests. 
    • "Borrower" searches for the borrowers that have placed requests

    Request Number, Bibliographic Info and Borrower

    At the Full Display of a request, you'll see an Edit Record button.

    Click this button to edit the record.

     

    When you close the record you'll go to the Requests List.

    ILL/Request Search

    At the full display of an ILL request or Request search, click Edit record to edit the request.

    Item/Article/Request for Information Request

    This allows staff to place an item, article or Request for Information request on the borrower's behalf. If there's a borrower 'in use' from Circulation, moving to the Request module will automatically this borrower's barcode in the 'Place request' page (similar to placing reservations).
