    Delete, deactivate or suspend collections
    In This Topic

    Delete a collection

    A collection cannot be deleted if there are items (including deactivated items) in the collection or if it is used in the scope of a policy.

    To delete a collection:

    1. Navigate to Maintenance > Circulation > Collections.
    2. Select a collection from the list.
    3. Click Delete, then click Yes at the confirmation prompt.
    4. Click Save or Save & Close. The collection is only deleted when changes are saved.

    Deactivate a collection

    A collection may be deactivated if there are no active links (items, policies, orders, invoices etc) to that collection.

    To deactivate a collection:

    1. Navigate to Maintenance > Circulation > Collections.
    2. Select a collection from the list.
    3. Click Deactivate, then click Yes at the confirmation prompt.
    4. Click Save or Save & Close.
    Deactivated collections will be hidden from the collection tree view by default, but may be included by clicking the Show Deactivated button in the action bar. If currently showing deactivated collections, click Hide Deactivated to hide them.

    Suspend a collection

    Suspending a Collection will prevent items from being added to the Collection. While a Collection is suspended, items cannot be moved into it using Item Maintenance or Bulk Item Maintenance.

    Suspended Collections are not suppressed from display at the OPAC. To suppress a Collection from OPAC display, do so via the Filters tab. For more detail, see the article on the hierarchy of OPAC suppression.

    A suspended Collection can be un-suspended later if required.

    To suspend a collection:

    1. From the Module Menu select Maintenance, Circulation and then Collections.
    2. Select a collection from the list.
    3. On the Details tab, click Suspend.
    4. Click Save or Save & Close. The collection is only suspended when you save your changes
