    Spydus Enquiry features in 2024R3
    In This Topic

    SavedList Search preset Type field to 'Any'

    The preset value for the Type dropdown in the SavedList Search page has been updated to 'Any'. This eases the search process for staff who frequently work with a combination of Public and Private SavedLists. 

    Previously the default preset value was Public, requiring staff to frequently switch the selection. 

    Enhancements to 'Download (CSV)' functionality

    This feature requires license version 11.7 or later.  

    Staff users can now specify the fields that are output in CSV files when exporting query results from the Enquiry module. Additionally, upon CSV file generation completion a message will be sent to the staff user who triggered it.

    Select required fields on Download (CSV)

    A new setting has been introduced in Staff General Parameters in the Maintenance module to Enable enhanced CSV download. Enabling this feature allows staff to customise the fields exported when using the Download (CSV) option in the Enquiry module.

    To enable this functionality:

    1. Navigate to Maintenance > General > Staff General Parameters.
    2. In the General section, set Enable enhanced CSV download to Yes.
    3. Click Save or Save & Close.

    Once enabled, upon clicking the Download (CSV) button at the Brief Display of search results (or at the Full Display of a Saved Query) the user will be presented with a field set to customise. Moving fields to the Output Fields (left side) will add that field to the CSV export. Moving fields to the Available Fields (right side) will remove them from the CSV export.

    images/2024R3_ENQ_CSV_FIELDS_thumb.png

    Click image to enlarge

    Once a field has been added to the Output Fields, it can be re-ordered by clicking on the  icon and dragging to the preferred position. Vertical order (top to bottom) in the Output Fields list will correspond to horizontal order (left to right) in the CSV file.

    By default, CSV field customisations are unique to the user and the record format (Borrower, Loan, Item, etc). These customisations are not stored in local cookies but in the database and so will follow the user between devices and locations.

    Message to user on CSV generation

    Once a user has clicked the Download button to begin generating a CSV, an on-screen prompt will advise that generation has begun and show a progress bar.

    Previously, if the screen tracking CSV progress was closed there was no way for staff to find the link to the file once the generation of the file was complete. Spydus now sends a message to the Spydus inbox of the user once CSV generation is complete, with a link to the file that is valid for 7 days.

    Instant Report, Print Set and SavedList Download enquiry options to generate a user message

    As with the CSV download feature above, previously when generating Instant Reports, using the Print Set function, or downloading a SavedList, if the screen tracking the progress of generating the file was closed there was no way for staff to find the link to the file once the generation of the file was complete.

    Spydus now sends a message to the Spydus inbox of the user when:

    • Generating an Instant Report from the Brief Display of a search result set
    • Clicking the Print Set button  at the Brief Display of a search result set
    • Using the Download option for a SavedList

    Once generation of the file or report is complete, a message will be sent to the inbox of the user who generated it.

    Option to control global expansion at search parameter level

    A new Enable global expansion setting has been added to Search Parameters Maintenance to enable staff to customise global search settings in Enquiry independent of default global search settings.

    This feature allows staff to to disable global expansion for specific search forms while keeping it enabled for others. e.g. Disabling global expansion for borrower search, while keeping holdings searches global.

    See the Spydus Maintenance features in 2024R3 page for more information on configuring this functionality.
