    Search tab
    In This Topic

    The parameters on this tab control the way that various aspects of OPAC searches will behave.

    Show search menu as an accordion

    If selected, the search menu will only allow one section to open at a time. Selecting another option in the menu will automatically collapse the other menus.

    Enable fuzzy search (advanced search)

    If selected, a radio button will show for fuzzy search on the quick search pages.

    Enable Autocomplete

    This feature requires Spydus 11.

    If selected, an autocomplete dropdown list will be displayed when search terms are entered in quick search at the OPAC. 

    Added values

    An facet has been added to All resources and Bibliographic searches at the OPAC. This allows users to filter records by the date that the record was created in Spydus.

    The default periods for the facet are:

    • 1 month / 30 days
    • 3 months / 90 days
    • 6 months / 180 days

    Adjacent image shows default values in the facet at the OPAC.

    The Added values table can be used to create new options for the facet, and edit or delete existing options.

    Add

    To add a new option for the Added facet:

    1. Navigate to Maintenance > OPAC & Enquiry > HTML Configurator > Search tab.  
    2. Click the Add Value button at the top of the Added values table.
    3. Enter a Description for the option, and the corresponding number of Days.
      • e.g. 1 year / 365 days
    4. Click OK to return to the Search tab, then click Save or Save & Close in the action bar.

     Edit

    To edit an existing option in the Added facet:

    1. Navigate to Maintenance > OPAC & Enquiry > HTML Configurator > Search tab.  
    2. Click the Edit  button at the end of the row in the Added values table.
    3. Edit the Description and/or Days value.
    4. Click OK to return to the Search tab, then click Save or Save & Close in the action bar.

    Delete

    To delete an existing option in the Added facet:

    • Navigate to Maintenance > OPAC & Enquiry > HTML Configurator > Search tab.  
    • Click the Delete button at the end of the row in the Added values table.
    • Click OK to confirm.
    • Click Save or Save & Close in the action bar.

    Enable live counts of records in 'Added' facet

    If this parameter is enabled, the facet will display a count of how many records are captured in the Added value period. If there are no records for a given value, that value will not be displayed in the facet. If no records at all were added in any of the defined periods, the facet will not be displayed.
